In this project I learned how to display images in different projections

Describe in your own words how you displayed the map in different projections using QGIS

In this project I learned how to display images in different projections. I started with a blank new template in QGIS then added a world map as a vector layer by uploading a .shp file obtained from naturalearthdata.com. I used the Indacatrix plugin to add the circles showing scale.

WGS84 Projection

This projection casues East-West disortion along the poles.

Pseudo Mercator

This projection causes size disortion at the poles, making Greenland appear as big as Africa.

Sphere Winkel

This projection preserves scale but caues some distance disortion.

World Cyndrilical Equal Area

This projection casues East-West disortion along the poles.

World Equidistant Conic

This conic projection causes the most disortion around the edges.

North Pole Azimuthal Equidistant

This conic projection causes the most disortion around the edges.

Aitoff Projection

This projection disorts scale and distance

World Miller Cyndrilical

This projectio preserves distance.

World Mollweide

This projection disorts distance.
